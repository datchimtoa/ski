# Trồng Cây Lụm Tiền

## Description
This project aims to create an engaging platform for planting virtual trees earning real money while learning about eco-friendliness and sustainability.

## Setup Instructions

### Prerequisites
- Node.js (>= 14.x)
- npm (>= 6.x)
- Git

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/datchimtoa/ski.git
   cd ski
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Configuration
1. Create a `.env` file in the root directory based on `.env.example`.
2. Update your database connection settings and API keys.

### Running the Application
To run the project in development mode:
```bash
npm start
```
To create a production build:
```bash
npm run build
```

## Architecture

### Overview
The application follows a client-server architecture.

### Components
- **Frontend**: Built with React, providing a responsive user interface.
- **Backend**: Node.js and Express server handling API requests.
- **Database**: MongoDB for storing user data and application state.

## Usage
Explore the platform by signing up, planting your first tree, and navigating through the various educational resources provided.
